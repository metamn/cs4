Breakpoints

Short description

The breakpoints used for responsive layouts.

Example

mobile

37.5em Screens with less than 600px width

laptop

64em Screens starting with 1024px width

desktop

90em Screens with more than 1440px width

Technical description

Around these breakpoints a set of responsive mixins are defined to cover all possible device sizes and make the design fully responsive.

Breakpoints X Axis

0 600.0 1024 1440

mobile

Mobile devices in both portrait and landscape mode

tablet

Tablets in both portrait and landscape mode

below-laptop

Mobile and tablet devices in both portrait and landscape mode

laptop-and-up

Everything above 1024px

desktop-and-up

For large screens above 1440px

Source code

Available code snippets

scss

assets/styles/atoms/_breakpoints.scss
$breakpoint-mobile: 37.5em; // Screens with less than 600px width
$breakpoint-laptop: 64em; // Screens starting with 1024px width
$breakpoint-desktop: 90em; // Screens with more than 1440px width



@mixin mobile {
  // Mobile devices in both portrait and landscape mode 
  @media only screen and (max-width: $breakpoint-mobile - 0.001) {
    @content
  }
}

@mixin tablet {
  // Tablets in both portrait and landscape mode 
  @media only screen and (min-width: $breakpoint-mobile) and (max-width: $breakpoint-laptop) {
    @content
  }
}

@mixin below-laptop {
  // Mobile and tablet devices in both portrait and landscape mode 
  @media only screen and (max-width: $breakpoint-laptop) {
    @content
  }
}

@mixin laptop-and-up {
  // Everything above 1024px 
  @media only screen and (min-width: $breakpoint-laptop + 001) {
    @content
  }
}

@mixin desktop-and-up {
  // For large screens above 1440px 
  @media only screen and (min-width: $breakpoint-desktop) {
    @content
  }
}
